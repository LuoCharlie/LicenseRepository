# 开源许可证知识库

#### 介绍
开源许可证知识库是用来展示许可证信息，展示许可证相关前沿技术文章的分享网站

#### 软件架构
- 前端采用JSP+BootStrap+JQuery
- 后端采用Servlet+JavaBean+Mysql


#### 已有功能

1.  管理员增删改查
2.  知识库、文章库模块
3.  许可证模块
4.  搜索引擎模块
